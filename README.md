### Описание переменных окружения

| ENV         | Описание  | required | default        |
|-------------|-----------|----------|----------------|
| DB_USER     | Логин БД  | true     |                |
| DB_PASSWORD | Пароль БД | true     |                |
| DB_HOST     | host:port |          | localhost:5557 |
| DB_NAME     | Имя БД    |          | db             |
